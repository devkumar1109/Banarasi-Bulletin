UP Beefs Up Security: New Battalions, New Campuses

2025-04-20T00:56:00+00:00

The Uttar Pradesh government, led by CM Yogi Adityanath, is prioritizing security infrastructure development. This involves constructing new campuses for the Uttar Pradesh Special Security Force (UPSSF) battalions. The 3rd Battalion will have residential and non-residential facilities built on 42.03 acres of land in Prayagraj's Arail area, divided between Deoria and Bhaipur villages. Additionally, a 4th UPSSF battalion will be established in Mathura. This initiative aligns with CM Yogi Adityanath's broader vision for enhanced security in the state.
