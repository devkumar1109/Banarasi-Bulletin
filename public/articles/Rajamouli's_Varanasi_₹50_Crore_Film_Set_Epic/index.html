Rajamouli's Varanasi: ₹50 Crore Film Set Epic

2025-06-18T15:12:00+00:00

SS Rajamouli is constructing a ₹50 crore set in Hyderabad for his upcoming film SSMB29, starring Mahesh Babu. The set is a grand replica of Varanasi, complete with ghats, temples, and the city's spiritual ambiance. This makes it the most expensive film set ever built in India, surpassing even the set of Sanjay Leela Bhansali's "Devdas". The set's scale and craftsmanship have impressed fans and industry insiders. Priyanka Chopra is also rumored to be part of the cast.
