UP Beefs Up Security with New Battalions

2025-04-20T00:56:00+00:00

The Uttar Pradesh government, led by CM Yogi Adityanath, is focusing on strengthening security infrastructure by constructing new campuses for the Uttar Pradesh Special Security Force (UPSSF). A 3rd Battalion will be established in Prayagraj with residential and non-residential facilities built on 42.03 acres of land in the Arail area. Additionally, a 4th Battalion will be set up in Mathura, reflecting CM Adityanath's commitment to enhancing security measures.
